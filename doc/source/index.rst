|PACKAGE_NAME|
==============

|PACKAGE_NAME| is an advanced computer algebra system.
Its capabilities range from calculus (integration and differentiation)
to abstract algebra. It can plot functions and has integrated help system.

FriCAS is a fork (2007) of the `Axiom <http://axiom-developer.org/>`_
computer algebra system.
